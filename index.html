1. Hệ thống thông tin quản trị vận hành và điều phối công việc

Loại dự án: Thực hành nghề nghiệp
Vai trò: Phân tích hệ thống, thiết kế cơ sở dữ liệu, phát triển chức năng
Công nghệ: Python Flask, Microsoft SQL Server, HTML/CSS, JavaScript, Bootstrap, Azure

Hệ thống web hỗ trợ số hóa quy trình quản lý văn bản, phân công nhiệm vụ và điều phối công việc tại đơn vị hành chính. Hệ thống giúp lưu trữ dữ liệu tập trung, hỗ trợ tra cứu nhanh, theo dõi tiến độ xử lý và lập báo cáo thống kê.

  • Quản lý công văn đi, công văn đến, nhiệm vụ, ý kiến kiến nghị và hội nghị cuộc họp.
  • Phân quyền người dùng theo vai trò: quản trị viên, lãnh đạo, văn thư, chuyên viên.
  • Thiết kế cơ sở dữ liệu SQL Server và xây dựng hệ thống web bằng Python Flask.
  • Triển khai thử nghiệm trên Azure App Service.

2. Hệ thống Quản lý Trung tâm Tin học KEY

Loại dự án: Đồ án Phân tích và Thiết kế Hướng Đối Tượng
Vai trò: Phân tích yêu cầu, thiết kế UML, thiết kế cơ sở dữ liệu, xây dựng chương trình minh họa
Công nghệ: C#, Windows Forms, .NET Framework, SQL Server 2022, Visual Studio 2022, Enterprise Architect

Dự án phân tích, thiết kế và xây dựng chương trình minh họa cho hệ thống quản lý trung tâm tin học KEY. Hệ thống hỗ trợ quản lý học viên, nhân viên, giảng viên, khóa học, lớp học, lịch học, tài chính, khuyến mãi, tài khoản người dùng, sao lưu dữ liệu và báo cáo thống kê.

  • Thiết kế Use Case, Activity Diagram, Sequence Diagram, Class Diagram, State Diagram.
  • Thiết kế cơ sở dữ liệu và ràng buộc dữ liệu bằng SQL Server.
  • Xây dựng chương trình minh họa bằng C# Windows Forms.
  • Áp dụng kiến trúc 3 tầng: Presentation - Business - Data.

3. PawFeed - Nền tảng kết nối thực phẩm dư cho trạm cứu hộ và động vật

Loại dự án: UI/UX Case Study | Mobile App Design | Figma Prototype
Vai trò: UX Research, UI Design, Wireframe, Prototype
Công cụ: Figma, User Flow, User Journey Map, Wireframe, Style Guide, Usability Testing

PawFeed là ý tưởng ứng dụng giúp kết nối người có thức ăn thú cưng dư thừa với các trạm cứu hộ, mái ấm động vật và tình nguyện viên. Thiết kế tập trung vào việc giảm lãng phí thức ăn, hỗ trợ trạm cứu hộ tiếp cận nguồn thực phẩm miễn phí, minh bạch và thuận tiện hơn.

  • Khảo sát người dùng, phân tích pain points và insight của người cho và người nhận.
  • Xây dựng persona, user scenario, user journey map và user flow.
  • Thiết kế wireframe, style guide và giao diện high-fidelity trên Figma.
  • Xây dựng prototype tương tác và đề xuất cải tiến sau kiểm thử người dùng.
